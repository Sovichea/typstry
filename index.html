About document scripts

Script assignments restrict each font to its intended Unicode script. Order is used as the tie-breaker when coverage overlaps; arrange the document's dominant script first. Scale is intended for small visual corrections; values beyond ±10% may vary between fonts. Typsastra writes portable Typst font assignments where possible and preserves source synchronization.
