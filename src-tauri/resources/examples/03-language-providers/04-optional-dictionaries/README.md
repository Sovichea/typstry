# Optional dictionaries

Purpose: test provider installation, removal, and unavailable document-script
assignments.

Expected behavior before installation: Document Typography shows the assigned
French and Arabic providers as unavailable, and those scripts receive no
spellcheck or completion.

Expected behavior after installation and assignment: each provider checks only
its configured script. Same-script languages such as English, French, and
Spanish never substitute for one another; choose one language for Latin in
Document Typography.

Tutorial: <https://github.com/Sovichea/typsastra/blob/main/docs/tutorials/LANGUAGE_PROVIDER_INSTALLATION.md>
